# Sanun Jara Elite

A React + TypeScript + Vite application with Express backend.

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deployment
See `DEPLOYMENT.md` for full deployment instructions.

- Frontend: Vercel (static build from `dist/`)
- Backend: Railway (`server/` folder)
- Database: MongoDB Atlas
